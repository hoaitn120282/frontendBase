## React Core

## Requirement

* [Node](https://nodejs.org/en/) >= 8.6.0
* [Yarn](https://yarnpkg.com/lang/en/) (recommend) >= 1.0.0

### Third-party

* [ant design](https://ant.design/)
* [reactstrap](https://reactstrap.github.io/)

### Setup

Install

```bash
yarn install
```

Start development

```bash
yarn start
```

Build

```bash
yarn build
```
